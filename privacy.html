1 Pengumpulan Data

Terima kasih telah menggunakan CitraKāra. Kami menghargai privasi Anda dan berkomitmen untuk melindunginya.

CitraKāra adalah aplikasi Desktop Offline (Lokal).

  • Kami TIDAK mengunggah foto Anda ke server manapun (Cloud).
  • Kami TIDAK mengumpulkan data pribadi, alamat IP, atau kebiasaan penggunaan Anda.
  • Semua proses penyortiran dan pemindahan file terjadi sepenuhnya di dalam hard drive/SSD komputer Anda sendiri.

2 Keamanan File & Non-Destructive

Software ini bekerja dengan prinsip Non-Destructive. Saat Anda melakukan aksi sortis/copy:

  • File asli di folder sumber atau Memory Card tidak akan diubah, dipindahkan, atau dihapus oleh sistem otomatis kami.
  • Aplikasi hanya melakukan operasi penyalinan (copy) ke folder tujuan.
  • Pengguna bertanggung jawab penuh atas pengelolaan backup data manual.
