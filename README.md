# citrakara

CitraKāra berasal dari bahasa Sanskerta: Citra (चित्र) berarti gambar, foto, citra visual. Kāra (कार) berarti pembuat, pengatur, atau pelaku suatu pekerjaan. Secara keseluruhan, CitraKāra dapat dimaknai sebagai “Pengolah atau Pengatur Gambar” — sangat sesuai untuk aplikasi yang berfokus pada manajemen foto.
